# Titen Diri v0.6

Versi pertama yang disiapkan untuk hidup sebagai bagian dari Bhumi Wanaprastha.

## Perubahan utama
- Empat lensa dipisahkan bahasa visualnya: Fisik pada tubuh, Meridian pada medan sekitar tubuh, Mental pada suasana latar, dan Cakra pada titik berwarna.
- Tubuh tidak lagi terlalu terang ketika semua lensa kuat.
- Jejak Titen memakai miniatur suasana watercolor berbasis CSS.
- Bank pengamatan diperluas menjadi 37 indikator.
- Lima pertanyaan jangkar selalu hadir, sepuluh pertanyaan lain dipilih secara konsisten untuk tanggal tersebut.
- Sintesis Corak Hari, Pantulan, dan Arah Laku memakai data pengamatan yang lebih kaya.
- Semua data tetap tersimpan lokal di browser.

## Menjalankan mandiri
Klik `start-server.bat`, lalu buka alamat yang ditampilkan.

## Integrasi Bhumi Wanaprastha
Salin seluruh folder ini ke `/apps/titen-diri/` dan tautkan dari konten aplikasi. Tidak memerlukan build khusus atau server tambahan.
