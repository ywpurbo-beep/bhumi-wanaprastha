# Template Konten Bhumi Wanaprastha

Salin template yang sesuai, lalu simpan salinannya di folder ruang tujuan. Jangan menulis kata `yaml` di atas front matter. File harus langsung dimulai dengan `---`.

## Template

- `01 Artikel.md`: Gagasan, Renungan, esai, kajian, dan panduan naratif.
- `02 Resep.md`: resep dan praktik berbentuk langkah.
- `03 Proyek.md`: karya arsitektur, interior, visual, publikasi, atau proyek lain.
- `04 Aplikasi.md`: aplikasi HTML dan perangkat interaktif.
- `05 Pengamatan.md`: catatan Kebun, observasi lapangan, dan eksperimen.

## Nilai `room`

`gagasan`, `renungan`, `laku`, `karya`, atau `kebun`.

## Aturan ID

Gunakan huruf kecil, angka bila perlu, dan tanda minus. Nama file sebaiknya sama dengan ID. ID tidak diubah setelah dipublikasikan.

## Tag dan relasi

Tag berisi topik, maksimal delapan. `related` berisi ID konten lain, bukan judulnya. Kosongkan dengan `[]` bila belum ada.

Setelah mengubah konten, jalankan `UPDATE CONTENT.bat`.
